# Quran Heaven

Welcome to **Quran Heaven**, a premium web application built with Django that offers Islamic content, including articles, fatwas, sheikh profiles, and more.

## Features

- **Modern & Premium UI/UX:** Designed with a stunning visual hierarchy featuring a Deep Emerald (`#1b4332`) and Gold (`#d4af37`) color palette.
- **Dynamic Content:** Browse through articles, categories, and series.
- **Fatwa Submission:** Users can easily submit fatwa requests.
- **Responsive Layout:** fully optimized for both desktop and mobile devices.
- **User Authentication:** Sign up, log in, and manage user profiles.

## Technologies Used

- **Backend:** Python, Django
- **Frontend:** HTML5, CSS3 (Bootstrap), JavaScript
- **Typography:** Google Fonts (Outfit and Amiri)
- **Database:** SQLite (default for development)

## Setup and Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Abdallh312/QuranHeaven.git
   cd QuranHeaven
   ```

2. **Create a virtual environment (optional but recommended):**
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows use `venv\Scripts\activate`
   ```

3. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

4. **Apply migrations:**
   ```bash
   python manage.py migrate
   ```

5. **Run the development server:**
   ```bash
   python manage.py runserver
   ```

6. **Access the application:**
   Open your browser and navigate to `http://127.0.0.1:8000/`.
